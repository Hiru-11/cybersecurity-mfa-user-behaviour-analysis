# Cybersecurity MFA Behaviour Analysis
This is an academic research project that explores Multi-Factor Authentication (MFA) push fatigue attacks and user behaviour patterns in digital authentication systems.

Focus:
1. Data cleaning and preprocessing of survey responses
2. Behavioural pattern analysis
3. Visualization of user response trends
4. Thematic categorization of findings

Methodology
1. Collected survey responses from Google Account users
2. Applied Python-based data analysis techniques
3. Used rule-based text classification for behavioural grouping
4. Generated visual insights using data visualization libraries

Technologies Used:

Python 

Files
1. data_cleaning.py
2. visualization.py
3. thematic_analysis.py

Note:

This project was completed as part of an academic coursework research assignment in cybersecurity and human-computer interaction (HCI).
